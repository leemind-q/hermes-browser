30-Second Quick Start

Test & run locally:

cd /mnt/c/Users/qqwer/Desktop/Hermes/hermes-browser
npm test            # 157 unit + 26 cowork + 46 eval (~3 sec)
npm start           # Launch Electron (Miraecle window)

Connect from external AI agents via HTTP bridge on port 8780:

TOKEN=$(curl -s http://127.0.0.1:8780/auth/token | jq -r .token)
curl -X POST http://127.0.0.1:8780/mcp/tool \\
  -H "Authorization: Bearer ***" \\
  -d '{"name":"cowork_list","args":{"path":"C:\\\\Users\\\\qqwer\\\\Hermes-Workspace\\\\demo-circuits"}}'
